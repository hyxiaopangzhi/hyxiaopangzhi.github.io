javascript ajax的总结

ajax

  • 定义: ajax 就是前端和后端进行交流的一门js技术

  • 作用:

    • 实现不刷新页面的情况下,更新局部页面

  • 分类(ajax 是前端和后端交流的技术,)

    • 同域请求:

      • 浏览器有同源策略(协议,域名,端口,)一致,就是同源,
      • 为什么有同源策略?
        • 为了用户安全;

      • 常规的ajax:

          1. 初始化一个ajax对象

            var xhr= new XMLHttpRequest();

          1. 创建连接

            xhr.open(“请求方式”,”请求地址”,”同步还是异步”);

          1. 发送数据

            xhr.send();

          1. 等待并拿到后台给我们的数据
            • 等待并且拿到后台给咱们的数据;
            • 通过检测xhr(ajax对象)的状态 来获取数据
            • xhr.readyState=4 ,说明了后台已经开始返回数据,
            • xhr.status==200 说明了 所有数据都传输完毕了
            • xhr.responseText 通过这个方法拿到数据

        • 请求方式 :

          • get 请求:
            • 数据都在地址栏中
            • 数据大小有限制
            • 传输速度快
            • 不安全
          • post 请求:
            • 数据在内部发送,相对安全
            • 传输速度相对较慢
            • 数据大小理论无限制
- 跨域请求(相对于你给你家人以外的人通电话);
    - 什么是跨域?
        > 协议,域名,端口,有一项和你当前的不一致,就是跨域;

    - 如何实现跨域:
          1. 动态创建一个带有 src 属性的标签, img, iframe , script(推荐使用)
          > var script=document.createElement("script");
          2. 让他的src属性等于你要请求的地址
          > script.src=你要请求的地址?callback=你要的方法
          3. 把你创建的标签放到文档中;
          > document.body.appendchild(script);
          4. 如果你需要对返回的数据操作,你得给他传递一个callback参数,callback=你要的方法
            - 页面上必须有一个 你要的方法,在这个方法中咱们就可以操作数据;
